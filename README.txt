InfinityLinks Windows Release

1. Copy .env.example to .env.
2. Open .env and set TMDB_API_KEY, TELEGRAM_BOT_TOKEN, TELEGRAM_CHANNEL_ID.
3. Run InfinityLinks.exe.
4. Open http://127.0.0.1:3000 in your browser.

The data folder stores the local SQLite database. Deleting the data folder resets the app.
